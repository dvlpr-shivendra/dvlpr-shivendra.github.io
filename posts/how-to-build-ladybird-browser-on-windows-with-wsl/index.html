How to Build Ladybird Browser on Windows with WSL

January 26, 2023 - One minute read - 207 words

Before we begin first you need to make sure that you have WSL 2 installed on your machine. WSL 2 is required to run GUI apps. If you have WSL 2 installed then you are ready to follow along.

We’ll use Ubuntu 22.04 because in Ubuntu 22.04 we can install GCC 12 with the apt package manager and we do not have to build it from the source.

To install Ubuntu 22.04 run this command in the powershell

wsl --install -d Ubuntu-22.04

Once the installation is done we would clone the Serenity OS source code because Ladybird is a part of the Serenity OS monorepo. To clone the source code run this command

git clone https://github.com/SerenityOS/serenity.git

Once the code of Serenity OS is cloned, We need to install the build dependencies. We would follow the steps described in serenity/Documentation/LadybirdBuildInstructions.md

Install build dependencies

sudo apt install build-essential cmake libgl1-mesa-dev ninja-build qt6-base-dev qt6-tools-dev-tools

Install GCC 12

sudo apt install gcc-12 g++-12

Generate build files

cmake -GNinja -S Ladybird -B Build/ladybird -D CMAKE_CXX_COMPILER /usr/bin/g++-12

Start build process

cmake --build Build/ladybird

Once the build precess is done we can run the Ladybird browser with this command

ninja -C Build/ladybird run
